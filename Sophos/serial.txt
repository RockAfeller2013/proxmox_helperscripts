How to set up the firewall
Install the downloaded image on your hardware or virtual environment.
Note: The installation overwrites the existing operating system and all files.
Connect the firewall's WAN interface (port 2) to your internet connection.
Connect an endpoint device to the LAN interface (port 1).
To access the setup assistant, go to the following IP address from the endpoint device: https://172.16.16.16:4444.
Note: The assistant may take a few minutes to start.
Follow the steps in the setup assistant.
