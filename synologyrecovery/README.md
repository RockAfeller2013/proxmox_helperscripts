# Synology Disk Recovery

## Process to recovery a Disk


1. Log a support case
2. Clean up room on Backup drive and check space required via dry run
3. Do a full photos back up and stop any more backups
5. rysycn files
6. run a check disk and unmount disk
7. Conduct full drive restore
8. copy files back

- File system errors occured on Volume 2. Back up your data and run a file system check.
